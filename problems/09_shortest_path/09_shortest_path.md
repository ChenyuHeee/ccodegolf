# 最短路径 (Shortest Path)

## 关键词
图论, Dijkstra算法, BFS, 最短路

## 难度
9/10 (地狱)

## 题面
给定一个有向带权图，求从节点 1 到节点 n 的最短路径长度。如果不存在路径，输出 -1。

输入格式：
- 第一行包含两个整数 n 和 m (2 ≤ n ≤ 100, 0 ≤ m ≤ 5000)，表示节点数和边数
- 接下来 m 行，每行三个整数 u, v, w (1 ≤ u, v ≤ n, 1 ≤ w ≤ 1000)，表示从节点 u 到节点 v 有一条权值为 w 的有向边

输出格式：
- 输出一个整数，表示从节点 1 到节点 n 的最短路径长度
- 如果不存在路径，输出 -1

## 示范样例

### 样例输入 1
```
4 5
1 2 1
1 3 4
2 3 2
2 4 5
3 4 1
```

### 样例输出 1
```
4
```

### 样例输入 2
```
3 2
1 2 1
2 3 2
```

### 样例输出 2
```
3
```

### 样例输入 3
```
3 1
2 3 1
```

### 样例输出 3
```
-1
```

## 提示
可以使用 Dijkstra 算法或 SPFA 算法。对于 code golf，考虑使用邻接矩阵和 Floyd-Warshall 算法可能代码更短，但要注意处理不可达的情况。
