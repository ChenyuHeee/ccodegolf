# 最长递增子序列 (Longest Increasing Subsequence)

## 关键词
动态规划, 二分查找, LIS算法

## 难度
7/10 (困难偏上)

## 题面
给定一个整数序列，求其最长严格递增子序列的长度。

输入格式：
- 第一行包含一个正整数 n (1 ≤ n ≤ 1000)
- 第二行包含 n 个整数 a_i (-10^9 ≤ a_i ≤ 10^9)

输出格式：
- 输出一个整数，表示最长严格递增子序列的长度

## 示范样例

### 样例输入 1
```
6
10 9 2 5 3 7
```

### 样例输出 1
```
3
```

### 样例输入 2
```
5
1 2 3 4 5
```

### 样例输出 2
```
5
```

### 样例输入 3
```
4
5 4 3 2
```

### 样例输出 3
```
1
```

## 提示
标准 O(n²) 动态规划算法较容易实现，但使用二分查找优化到 O(n log n) 的算法代码更短且更高效。
