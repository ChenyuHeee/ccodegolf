# 斐波那契取模 (Fibonacci Modulo)

## 关键词
斐波那契数列, 动态规划, 矩阵快速幂, 取模运算

## 难度
5/10 (中等偏难)

## 题面
计算第 n 个斐波那契数对 1000000007 取模的结果。斐波那契数列定义为：
- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) (n ≥ 2)

输入格式：
- 第一行包含一个非负整数 n (0 ≤ n ≤ 10^6)

输出格式：
- 输出 F(n) mod 1000000007 的结果

## 示范样例

### 样例输入 1
```
10
```

### 样例输出 1
```
55
```

### 样例输入 2
```
100
```

### 样例输出 2
```
687995182
```

### 样例输入 3
```
0
```

### 样例输出 3
```
0
```
