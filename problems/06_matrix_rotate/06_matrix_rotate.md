# 矩阵旋转 (Matrix Rotation)

## 关键词
矩阵操作, 空间优化, 就地算法

## 难度
6/10 (困难)

## 题面
给定一个 n×n 的矩阵，将其顺时针旋转 90 度后输出。

输入格式：
- 第一行包含一个正整数 n (1 ≤ n ≤ 100)
- 接下来 n 行，每行 n 个整数，表示矩阵的元素 (-1000 ≤ element ≤ 1000)

输出格式：
- 输出旋转后的矩阵，每行 n 个整数，用空格分隔

## 示范样例

### 样例输入 1
```
3
1 2 3
4 5 6
7 8 9
```

### 样例输出 1
```
7 4 1
8 5 2
9 6 3
```

### 样例输入 2
```
2
1 2
3 4
```

### 样例输出 2
```
3 1
4 2
```
