# 数字根

## 关键词
数学技巧, 循环优化, 模运算

## 难度
1/10 (入门)

## 题面
给定一个正整数 n，不断将其各位数字相加，直到得到一个一位数为止。这个一位数称为 n 的数字根。

例如：38 → 3+8=11 → 1+1=2，所以 38 的数字根是 2。

输入格式：
- 第一行包含一个正整数 n (1 ≤ n ≤ 10^9)

输出格式：
- 输出一个整数，表示 n 的数字根

## 示范样例

### 样例输入 1
```
38
```

### 样例输出 1
```
2
```

### 样例输入 2
```
123456789
```

### 样例输出 2
```
9
```

### 样例输入 3
```
999
```

### 样例输出 3
```
9
```
