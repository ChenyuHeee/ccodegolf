# 质数判断 (Prime Checker)

## 关键词
质数, 数论, 算法优化

## 难度
3/10 (中等偏易)

## 题面
判断一个正整数是否为质数。质数是指大于 1 且只能被 1 和自身整除的整数。

输入格式：
- 第一行包含一个正整数 n (2 ≤ n ≤ 1000000)

输出格式：
- 如果 n 是质数，输出 "YES"
- 如果 n 不是质数，输出 "NO"

## 示范样例

### 样例输入 1
```
7
```

### 样例输出 1
```
YES
```

### 样例输入 2
```
12
```

### 样例输出 2
```
NO
```

### 样例输入 3
```
97
```

### 样例输出 3
```
YES
```
