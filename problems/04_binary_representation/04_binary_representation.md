# 二进制表示 (Binary Representation)

## 关键词
进制转换, 位运算, 递归

## 难度
4/10 (中等)

## 题面
给定一个非负整数，输出其二进制表示（不含前导零，但 0 输出为 "0"）。

输入格式：
- 第一行包含一个非负整数 n (0 ≤ n ≤ 10^9)

输出格式：
- 输出 n 的二进制表示

## 示范样例

### 样例输入 1
```
5
```

### 样例输出 1
```
101
```

### 样例输入 2
```
42
```

### 样例输出 2
```
101010
```

### 样例输入 3
```
0
```

### 样例输出 3
```
0
```
