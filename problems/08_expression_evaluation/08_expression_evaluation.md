# 表达式求值 (Expression Evaluation)

## 关键词
栈, 表达式解析, 递归下降, 运算符优先级

## 难度
8/10 (很困难)

## 题面
给定一个只包含加法(+)、减法(-)、乘法(*)、除法(/)和括号的算术表达式，计算其结果（整数除法，向下取整）。

输入格式：
- 一行字符串，表示算术表达式，长度不超过 1000
- 表达式中只包含数字、运算符 +、-、*、/、左括号 (、右括号 )
- 保证表达式合法且不会除以零
- 所有数字都是非负整数

输出格式：
- 输出表达式的计算结果

## 示范样例

### 样例输入 1
```
3+5*2
```

### 样例输出 1
```
13
```

### 样例输入 2
```
(1+2)*3
```

### 样例输出 2
```
9
```

### 样例输入 3
```
10/3+5
```

### 样例输出 3
```
8
```
