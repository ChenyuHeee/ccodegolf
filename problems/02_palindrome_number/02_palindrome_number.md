# 回文数判断

## 关键词
回文, 数学运算, 逻辑判断

## 难度
2/10 (简单)

## 题面
判断一个整数是否为回文数。回文数是指正着读和反着读都一样的整数。

例如：121、12321 是回文数，123、-121 不是回文数。

注意：负数不是回文数。

输入格式：
- 第一行包含一个整数 n (-10^9 ≤ n ≤ 10^9)

输出格式：
- 如果 n 是回文数，输出 "YES"
- 如果 n 不是回文数，输出 "NO"

## 示范样例

### 样例输入 1
```
121
```

### 样例输出 1
```
YES
```

### 样例输入 2
```
-121
```

### 样例输出 2
```
NO
```

### 样例输入 3
```
12321
```

### 样例输出 3
```
YES
```

### 样例输入 4
```
123
```

### 样例输出 4
```
NO
```
